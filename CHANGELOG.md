# Changelog

Der `Lernmonitor` benutzt [semantic versioning](https://semver.org/), d.h. Versionen werden 
nach MAJOR.MINOR.PATCH-beta.X benannt. In diesem Changelog werden alle Änderungen
seit release der ersten Version katalogisiert, um Änderungen an der App nachverfolgen zu können.
In den ersten Monaten werden Änderungen vermutlich sofort released, vor allem wenn es sich
um kleinere Bugs handelt (konventionell als sometimes "patch" (`+0.0.1`) release.)
